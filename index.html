Course Overview

Software engineering is a disciplined approach to software development grounded in a core set of concepts, techniques, and tools. Software engineers must learn and practice these fundamentals, especially when collaborating with others on software projects. This course provides a foundation in software engineering for higher-level courses and co-op positions, including an introduction to the effective and responsible use of generative AI and agentic AI in software development practice.

This course requires reliable, consistent access to a desktop or laptop with at least 8GB of RAM and at least 250 GB of disk space. Information on student numbers, student grades, and submitted work will be stored in file systems and computers under the physical control of UVic.

Course Objectives and Learning Outcomes

Students successfully completing SENG 265 will be able to:

  • Describe basic software-engineering concepts
  • Construct solutions for problems using the programming language C
  • Use scripting languages, libraries and frameworks to improve software productivity.
  • Employ the bash-shell and the Linux operating system in the work of developing software
  • Construct solutions for problems using the programming language Python 3
  • Learn incremental software development techniques
  • Learn object-oriented programming using Python 3
  • Learn Python 3 decorators
  • Generate C and Python code using generative AI
  • Organize development work using software-configuration management tools such as git and make
  • Explain the purpose of a software-development process
  • Investigate the dynamic behavior of C programs
  • Identify and outline the test cases needed to give confidence in the correctness of a software-development artifact
