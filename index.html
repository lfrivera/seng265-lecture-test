SENG 265: Software Development Methods

Summer 2026 (1.5 Units)

Master the fundamentals of software engineering with hands-on experience in Linux, Git, C, Python, and modern development tools.

Course Overview

Software engineering is a disciplined approach to software development grounded in a core set of concepts, techniques, and tools. This course provides a foundation in software engineering for higher-level courses and co-op positions, including an introduction to the effective and responsible use of generative AI and agentic AI in software development practice.

Learning Outcomes

Students successfully completing SENG 265 will be able to:

  • Describe basic software-engineering concepts
  • Construct solutions for problems using C and Python 3
  • Use scripting languages, libraries and frameworks to improve productivity
  • Employ the bash-shell and Linux operating system for software development
  • Learn incremental software development techniques and object-oriented programming
  • Generate code using generative AI
  • Organize development work using software-configuration management tools
  • Explain the purpose of a software-development process
  • Identify and outline test cases for software artifacts
